# Alcance del Bloque 1

## Incluido

- Proyecto Visual Studio listo.
- Lectura de imágenes por carpetas.
- Preprocesamiento básico.
- Tensor y Matrix.
- Dense, ReLU, SoftmaxCrossEntropy.
- SGD y accuracy.
- Pruebas internas.
- Resultados en CSV/TXT.

## No incluido todavía

- Conv2D.
- MaxPool2D.
- Flatten.
- CNN completa.
- Backpropagation convolucional.
- OpenMP.
- Matriz de confusión final.
- Entrenamiento completo con todas las imágenes.

Este bloque está diseñado para que esas piezas se agreguen sin rehacer la arquitectura.
